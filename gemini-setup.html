1 Create a Google Cloud Project

You'll need a Google Cloud project to access the Gemini API. This is free and takes just a few minutes.

  1. Go to Google Cloud Console
  2. Click "Select a Project" at the top
  3. Click "NEW PROJECT"
  4. Enter a project name (e.g., "Rupiya AI")
  5. Click "CREATE"
üîí Security Note: Your Google Cloud project is private and only you can access it. No one else can see your API key.
2 Enable the Gemini API

Enable the Generative Language API in your Google Cloud project.

  1. Make sure your new project is selected
  2. Go to Generative Language API
  3. Click "ENABLE"
  4. Wait for the API to be enabled (usually takes a few seconds)
